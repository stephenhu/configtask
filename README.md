# Configly

Rakefile extension to automatically generate configuration files for
web sites.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'configly'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install configly

## Usage

rake config:create
